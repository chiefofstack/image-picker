Instructions:
  1. Enter a valid email address
  2. Hit save to store the image
  3. The list will update automatically
  4. Previously used emails will be added to the email dropdown
  5. You can click refresh to load a new image
  6. You can save as many images as you want
  7. You can use as many emails as you want
